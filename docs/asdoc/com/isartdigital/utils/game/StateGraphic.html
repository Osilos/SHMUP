Classpublic class StateGraphic
InheritanceStateGraphic GameObject flash.display.Sprite

Classe de base des objets interactifs ayant plusieurs états graphiques Elle gère la représentation graphique (anim) et les conteneurs utiles au gamePlay (box) qui peuvent être de simples boites de collision ou beaucoup plus suivant l'implémentation faite par le développeur dans les classes filles
